# CSS Styling Exercise – Design Mockup Replication

An external CSS stylesheet built to replicate a provided design mockup as closely as possible, 
demonstrating a wide range of core CSS styling techniques.

## Requirements Met
- Background color matches the mockup
- Header sizes and alignment match
- Second paragraph replicates exact width, margin, padding, border, and line-height
- Fourth and fifth paragraphs use correct text colors
- Link colors match mockup, with the W3C Schools link styled differently from list item links
- Ordered list uses two-digit numbering
- Table header and cells have distinct background colors
- Table borders apply to all cells with overlapping (collapsed) borders

## Files
- `index.html` — provided HTML file with linked stylesheet
- `styles.css` — external CSS file written to match the mockup

## How to View
Open `index.html` in any web browser — no setup required.

## Concepts Used
- External CSS stylesheets
- Box model (margin, padding, border)
- Typography and line-height
- Background and text color styling
- Anchor and link pseudo-class styling
- CSS list styling with counter formatting
- Table styling with border-collapse
- CSS specificity and selector targeting
